Accelerating materials discovery with machine learning requires more than individual models — it demands robust, reproducible infrastructure that connects raw data, feature engineering, model training, evaluation, and deployment into a coherent workflow. This project builds an end-to-end ML platform purpose-built for materials property prediction and virtual screening.

The platform supports multiple featurization backends (RDKit, Matminer, custom graph encoders), automated model selection and hyperparameter tuning via Optuna, and MLflow-based experiment tracking. A modular pipeline architecture allows plug-and-play substitution of featurizers and models, making the system applicable across diverse materials classes — from small organic molecules to crystalline inorganic solids. A lightweight web interface exposes model predictions for interactive exploration.

Deployed on property prediction tasks (formation energy, band gap, solubility, mechanical moduli), the platform reduces the time from raw dataset to production-ready predictions from weeks to hours, demonstrating the value of ML engineering discipline in scientific research contexts.
