Planning a trip involves balancing dozens of competing constraints — budget, timing, travel style, interest categories, logistics — a task where large language models and agentic AI systems can add substantial value. TravelAI is a conversational planning agent that constructs personalized, day-by-day itineraries through natural dialogue, adapting recommendations dynamically based on user feedback.

The agent is built on an LLM backbone with tool-use capabilities, integrating real-time data sources for flights, accommodations, and points of interest. A preference model, updated incrementally through the conversation, drives personalized ranking of options. Constraint satisfaction logic ensures scheduling feasibility (opening hours, travel times, capacity). An iterative refinement loop allows users to tweak individual days without restarting the entire plan.

TravelAI reflects a broader interest in deploying agentic AI systems for complex planning tasks — demonstrating how careful system design can transform a capable base model into a reliable, user-aligned assistant for real-world decision-making.
